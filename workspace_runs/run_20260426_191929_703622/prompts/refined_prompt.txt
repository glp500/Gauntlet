# Task Objective
Analyze the provided sales and inventory dataset to produce a clear summary and generate at least one visualization (figure) that explains the key findings of the analysis.

# Input Dataset Summary
The analysis will be performed on a single CSV file named `Warehouse_and_Retail_Sales.csv`.
*   **Rows:** 307,645
*   **Columns:** YEAR, MONTH, SUPPLIER, ITEM CODE, ITEM DESCRIPTION, ITEM TYPE, RETAIL SALES, RETAIL TRANSFERS, WAREHOUSE SALES.
*   **Data Types:** Numerical fields (RETAIL SALES, RETAIL TRANSFERS, WAREHOUSE SALES) are float64. Date fields (YEAR, MONTH) are int64. Categorical fields (SUPPLIER, ITEM DESCRIPTION, ITEM TYPE) are string.

# Assumptions From Schema
1.  The analysis should focus on aggregating and summarizing the sales, transfers, and warehouse data across the provided dimensions (Year, Month, Supplier, Item Type).
2.  'RETAIL SALES', 'RETAIL TRANSFERS', and 'WAREHOUSE SALES' are the primary numerical metrics for aggregation.
3.  The analysis should consider relationships between sales, transfers, and warehouse stock.

# Required Outputs
1.  A clear, descriptive summary of the dataset's key findings.
2.  At least one figure (visualization) created using pandas and matplotlib to visually explain a significant result derived from the analysis.

# Code Constraints
1.  The generated code must implement functions only within the following files: `data_loader.py`, `preprocessing.py`, `analysis.py`, and `figures.py`.
2.  Only the pandas and matplotlib libraries are permitted.
3.  No external network access is allowed.
4.  No shell commands are allowed.
5.  The logic must be explicit and highly readable.
